# SEO e indexação no Google

Arquivos preparados para o domínio:

- https://psicologavanessamiranda.com.br/

## Arquivos incluídos

- `index.html` com title, description, canonical, Open Graph, Twitter Card e dados estruturados JSON-LD.
- `robots.txt` liberando rastreamento e indicando o sitemap.
- `sitemap.xml` com a URL principal do site.
- `site.webmanifest` para identidade básica do site em dispositivos.
- `.nojekyll` para publicação estática correta no GitHub Pages.

## Como acelerar a indexação

1. Suba todos os arquivos para a raiz do repositório no GitHub.
2. Aguarde o GitHub Pages publicar com HTTPS.
3. Acesse o Google Search Console.
4. Adicione a propriedade `psicologavanessamiranda.com.br`.
5. Verifique a propriedade preferencialmente pelo DNS no Cloudflare.
6. Envie o sitemap: `https://psicologavanessamiranda.com.br/sitemap.xml`
7. Use a inspeção de URL para a página: `https://psicologavanessamiranda.com.br/`
8. Clique em Solicitar indexação.

## Observação importante

O Google não garante indexação imediata. Estes arquivos ajudam o Google a rastrear e entender o site corretamente, mas a velocidade final depende do próprio Google.
