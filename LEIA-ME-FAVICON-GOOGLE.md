# Favicon / ícone da logo no Google

Esta versão inclui novos ícones para o Google exibir a logo no resultado de busca.

## Arquivos adicionados/ajustados

- `/favicon.ico`
- `/assets/img/favicon-48x48.png`
- `/assets/img/favicon-96x96.png`
- `/assets/img/favicon-192x192.png`
- `/assets/img/favicon-512x512.png`
- `/assets/img/apple-touch-icon.png`
- `/site.webmanifest`
- `index.html` com links corretos no `<head>`

## Depois de subir para o GitHub

1. Aguarde o GitHub Pages publicar.
2. Teste se estes links abrem:
   - `https://psicologavanessamiranda.com.br/favicon.ico`
   - `https://psicologavanessamiranda.com.br/assets/img/favicon-48x48.png`
3. No Google Search Console, use "Inspeção de URL" em:
   - `https://psicologavanessamiranda.com.br/`
4. Clique em "Solicitar indexação".

## Importante

O Google pode demorar alguns dias ou até algumas semanas para trocar o globo pela logo nos resultados, porque o favicon fica em cache e depende de novo rastreamento.
